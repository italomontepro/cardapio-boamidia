# Plan 03-04: Products CRUD + Photos + Accordion

## Objective

Complete the cardapio experience by layering products into the category structure built in Plan 03-03:
- Append product zod schema to `src/lib/catalog/schema.ts` and product Server Actions to `src/lib/catalog/actions.ts`.
- `upsertProduct(FormData)` (insert/update + Supabase Storage photo upload via service_role + getPublicUrl), `deleteProduct`, `moveProductUp`/`moveProductDown` (sort_order swap within category) — scoped to `getCurrentAdmin().restaurantId` (CTLG-01, CTLG-02, CTLG-03, CTLG-04).
- Evolve `/painel/cardapio/page.tsx` to fetch categories WITH products and render `CardapioAccordion` (shadcn Accordion) showing products inline; "+" to add a product inside each category (D-10, D-11, CTLG-05).
- Product form Dialog: name, description (Textarea), price (text inputMode decimal), is_featured (Checkbox), photo (file input + preview) (D-13, D-18).
- Product delete AlertDialog (D-14).

Purpose: Finishes the phase's most complex surface — file upload + accordion + product CRUD — completing the restaurant admin's menu-building toolset.
Output: catalog lib (products), accordion-based cardapio page, product components, full verify-catalog.ts including a live Storage upload assertion.

## must_haves
- [ ] `upsertProduct` inserts/updates a product, uploads the file (when present, size>0) to `product-images` at `{restaurantId}/{productId}/photo.{ext}` with `{ upsert: true }`, and writes the public URL to `products.image_url`
- [ ] Price input like "29,90" is normalized to numeric 29.90; invalid/negative price rejected
- [ ] `isFeatured` checkbox persists to `products.is_featured`
- [ ] `moveProductUp`/`moveProductDown` swap sort_order atomically within the same category
- [ ] cardapio page renders a shadcn `Accordion`; products appear inside their category
- [ ] `npx tsx scripts/verify-catalog.ts` asserts CTLG-01/02/03/04(products) and exits 0

           ```typescript
           const file = formData.get('photo') as File | null
           if (file && file.size > 0) {
             if (file.size > 5 * 1024 * 1024) return { error: { photo: ['Foto deve ter menos de 5 MB.'] } }
             if (!['image/jpeg', 'image/png', 'image/webp'].includes(file.type)) {
               return { error: { photo: ['Formato inválido. Use JPG, PNG ou WebP.'] } }
             }
             const ext = (file.name.split('.').pop() || 'jpg').toLowerCase()
             const path = `${admin.restaurantId}/${product.id}/photo.${ext}`
             const supabaseAdmin = createAdminClient()
             const { error: uploadError } = await supabaseAdmin.storage
               .from('product-images')
               .upload(path, file, { upsert: true, contentType: file.type })
             if (uploadError) return { error: { photo: ['Erro ao fazer upload da foto.'] } }
             const { data: { publicUrl } } = supabaseAdmin.storage.from('product-images').getPublicUrl(path)
             await db.update(products).set({ imageUrl: publicUrl }).where(eq(products.id, product.id))
           }
           ```
