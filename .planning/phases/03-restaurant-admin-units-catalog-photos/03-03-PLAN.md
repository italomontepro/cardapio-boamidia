# Plan 03-03: Categories CRUD + Reorder

## Objective

Build the menu category management on `/painel/cardapio` (categories only — products are layered on in Plan 04):
- Create `src/lib/catalog/schema.ts` + `src/lib/catalog/actions.ts` (categories portion; Plan 04 appends product actions to the same files).
- `createCategory` (sort_order = MAX+1), `updateCategory` (name), `deleteCategory` (cascade), `moveCategoryUp` / `moveCategoryDown` (atomic sort_order swap via `db.transaction()`) — all scoped to `getCurrentAdmin().restaurantId` (UNIT-03, CTLG-04).
- `/painel/cardapio` page listing categories in sort_order with ↑/↓/Editar/Remover actions; "Nova Categoria" button at top (D-11, D-12).
- Category delete AlertDialog warns products will be removed by cascade (D-14).

Purpose: Establishes the cardapio page shell and category CRUD that Plan 04 wraps in an accordion with products inside.
Output: catalog lib (categories), cardapio page + category components, extended verify-catalog.ts.

## must_haves
- [ ] `createCategory` sets `sort_order = COALESCE(MAX(sort_order),-1)+1` scoped to restaurant
- [ ] `moveCategoryUp`/`moveCategoryDown` swap sort_order atomically inside `db.transaction()`
- [ ] `deleteCategory` removes the row (products cascade) scoped to restaurant
- [ ] `/painel/cardapio` lists categories in ascending sort_order
- [ ] `npx tsx scripts/verify-catalog.ts` asserts UNIT-03 + CTLG-04 (category reorder) and exits 0

    1. Create `src/lib/catalog/schema.ts`:
       ```typescript
       import { z } from 'zod'

       export const upsertCategorySchema = z.object({
         id: z.string().uuid().optional(),
         name: z.string().trim().min(1, 'Nome é obrigatório.').max(80, 'Nome deve ter no máximo 80 caracteres.'),
       })
       ```

       (Product schema is appended to this file in Plan 04 — leave room.)
    2. Create `src/lib/catalog/actions.ts` with `'use server'`. Imports: `db` from `@/db`; `{ categories }` from `@/db/schema`; `{ eq, and, lt, gt, desc, asc, sql }` from `drizzle-orm`; `revalidatePath` from `next/cache`; `getCurrentAdmin` from `@/lib/auth/session`; `{ upsertCategorySchema }` from `./schema`.
       Define a local helper `function revalidateCardapio() { try { revalidatePath('/painel/cardapio') } catch {} }`.
       Implement (each `{ success: true }` | `{ error: ... }`):
       - `createCategory(input: { name: string })`:
         - `const admin = await getCurrentAdmin(); if (!admin?.restaurantId) return { error: { _form: ['Não autorizado.'] } }`
         - `const parsed = upsertCategorySchema.safeParse(input); if (!parsed.success) return { error: parsed.error.flatten().fieldErrors as Record<string,string[]> }`
         - `const [maxRow] = await db.select({ max: sql<number>\`COALESCE(MAX(sort_order), -1)\` }).from(categories).where(eq(categories.restaurantId, admin.restaurantId))`
         - `const nextSortOrder = (maxRow?.max ?? -1) + 1`
         - `await db.insert(categories).values({ restaurantId: admin.restaurantId, name: parsed.data.name, sortOrder: nextSortOrder })`
         - `revalidateCardapio(); return { success: true }`
       - `updateCategory(input)`: require `parsed.data.id`; `db.update(categories).set({ name }).where(and(eq(categories.id, id), eq(categories.restaurantId, admin.restaurantId)))`; revalidate; success.
       - `deleteCategory(id: string)`: scope to admin restaurant; `db.delete(categories).where(and(eq(categories.id, id), eq(categories.restaurantId, admin.restaurantId)))` (products cascade via FK); revalidate; success.
       - `moveCategoryUp(categoryId: string)`: get admin; `const [current] = await db.select().from(categories).where(and(eq(categories.id, categoryId), eq(categories.restaurantId, admin.restaurantId)))`; if `!current` return `{ error: { _form: ['Categoria não encontrada.'] } }`; `const [neighbor] = await db.select().from(categories).where(and(eq(categories.restaurantId, admin.restaurantId), lt(categories.sortOrder, current.sortOrder))).orderBy(desc(categories.sortOrder)).limit(1)`; if `!neighbor` return `{ success: true }` (already top); then:
         ```typescript
         await db.transaction(async (tx) => {
           await tx.update(categories).set({ sortOrder: neighbor.sortOrder }).where(eq(categories.id, current.id))
           await tx.update(categories).set({ sortOrder: current.sortOrder }).where(eq(categories.id, neighbor.id))
         })
         ```
         revalidate; success.
       - `moveCategoryDown(categoryId: string)`: same but neighbor uses `gt(categories.sortOrder, current.sortOrder)` ordered `asc(categories.sortOrder)`.
       - Wrap DB writes in try/catch returning `{ error: { _form: ['Não foi possível salvar. Tente novamente.'] } }`.
  </action>
